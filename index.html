FAQ

Como funciona?

Ao contrário de outras soluções que utilizam bancos de imagens pré-existentes, a Midjourney cria imagens únicas e personalizadas para cada usuário, permitindo que você obtenha exatamente o que precisa para o seu projeto. Basta escolher o estilo, a cor e o tema da imagem que você deseja e a plataforma se encarregará de produzir uma imagem exclusiva, que atenda às suas necessidades e especificações.

Quais as vantagens?

Com Midjourney, você pode economizar tempo e dinheiro, evitando a contratação de fotógrafos ou designers gráficos para criar suas imagens personalizadas. Além disso, a plataforma é fácil de usar e acessível, permitindo que usuários de todos os níveis de habilidade possam criar imagens impressionantes com facilidade.

Como começar a usar?

Basta acessar o discord, utilizar alguns dos canais de iniciante e interajir com o Midjourney no Discord. Os comandos são usados para criar imagens, alterar configurações padrão, monitorar informações do usuário e executar outras tarefas úteis. O comando /imagine gera uma imagem exclusiva a partir de uma breve descrição de texto (conhecida como Prompt ).
